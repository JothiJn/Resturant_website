# Rocky Restaurant | Luxury Dining Experience

A premium, full-stack restaurant website built with Next.js 15, offering a seamless reservation system, event inquiries, and a personalized user dashboard.

## 🌟 Features

- **Luxury Aesthetics**: Modern, responsive design with glassmorphism and smooth animations.
- **Secure Authentication**: Google OAuth integration via Supabase.
- **Reservation System**: Real-time reservation booking and management.
- **Event Inquiries**: Specialized inquiries for private events and catering.
- **User Dashboard**: Personalized area for users to track their bookings.
- **Admin Management**: Streamlined interface for managing restaurant operations.

## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS 4
- **Database & Auth**: Supabase
- **Animations**: Framer Motion
- **Validation**: Zod
- **Type Safety**: TypeScript

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/restaurant-website.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up environment variables**:
   Create a `.env.local` file with the following:
   ```env
   NEXT_PUBLIC_SUPABASE_URL=your-supabase-url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your-supabase-anon-key
   NEXT_PUBLIC_SITE_URL=http://localhost:3000
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```

## 🌐 Deployment

The project is optimized for deployment on **Vercel**. Ensure you add the environment variables listed above in your Vercel project settings.
